アプリケーション名：AquaList
目的：
全国の水族館を掲載し、誰もが水族館を簡単に調べられるようにする。
管理者が訪れたことのある水族館を記録する。

機能
水族館を新規作成・編集できる。
登録された水族館を一覧・詳細画面で閲覧できる。
水族館詳細画面でユーザーがコメントできる。
水族館詳細画面でユーザーが星マークで水族館を評価できる。
水族館詳細画面でユーザーがハートマークで水族館をお気に入り登録できる。
ユーザー登録ができる。
ユーザーがログイン・ログアウトできる。
ユーザーから管理者に問合せができる。
